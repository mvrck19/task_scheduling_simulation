# Task Scheduling Simulation with HEFT Algorithm

This project simulates task scheduling on virtual machines using the HEFT (Heterogeneous Earliest Finish Time) algorithm. The simulation is designed to model a workflow of tasks and virtual machines, and it includes a modular structure that allows for the extension and simulation of other scheduling algorithms.
